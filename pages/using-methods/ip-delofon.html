Yealink SIP-T30 Delofon — удобство для вашего бизнеса

Делофон — это IP‑телефоны, которые уже подключены к виртуальной телефонии Делофон. IP‑телефон Yealink SIP‑T30 Delofon со встроенной телефонией может быть использован в различных сферах бизнеса, от малого офиса до крупной компании. Для использования данного устройства необходимо всего лишь подключить его к сети интернет.

Кроме того, Делофон и компьютер можно подключить к интернету одним кабелем. В каждом Yealink ‑T30 Delofon есть разъём для подключения ПК. Подключайте ПК к телефону имеющимся LAN-кабелем, а телефон к роутеру кабелем, который есть в комплекте.

Также, к телефону из комплекта можно подключить ещё одного провайдера IP‑телефонии. Yealink SIP‑T30 Delofon поддерживает параллельную работу сразу с двумя SIP‑аккаунтами. Можно одновременно использовать Делофон и другого провайдера.

Помимо перечисленных преимуществ, IP-Делофон имеет множество функций, которые помогают в повышении производительности офиса: распределение и переадресация звонков, голосовое меню, запись разговоров и т.д.
